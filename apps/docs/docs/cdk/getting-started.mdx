# Getting started

`@rx-angular/cdk` is a toolkit of independent building blocks. You install it once and import only the blocks you use.

## Install

```sh
npm install @rx-angular/cdk
```

## Import a building block

Each block lives in its own entry point. For example, you import the render strategies provider from the render strategies entry point.

```ts
import { RxStrategyProvider } from '@rx-angular/cdk/render-strategies';
```

## Where to go next

- Browse every building block on the [Overview](./overview.mdx).
- Pick a block such as [Render strategies](./render-strategies/render-strategies.mdx) and read its page.
